# FINTECH-EMPLOYEE-PROJECT-MANAGEMENT-SYSTEM
This project is a comprehensive SQL data analysis case study focused on evaluating workforce structure, project assignments, and departmental performance within a fictitious company (Fintech).

The organization develops innovative financial solutions such as:

Mobile payment platforms

Fraud detection systems

Robo-advisors

Digital wallets

Regulatory compliance solutions

The goal of this project is to extract meaningful insights about workforce performance, project management, and departmental efficiency using SQL queries and analytical techniques.

# Dataset Description

The dataset consists of three relational tables representing employees, projects, and project-department mappings.

 Table 1: employees

Stores employee demographics, employment history, and performance evaluations.

Table 2: projects

Contains information about projects and assigned employees.

Table 3: project_departments

Maps projects to their lead departments.

# Project Objectives

This project aims to:

Analyze employee performance and compensation

Track project progress and resource allocation

Evaluate departmental workload and efficiency

Practice advanced SQL querying techniques

Strengthen real-world data analysis problem-solving skills

# Data Preparation & Cleaning
